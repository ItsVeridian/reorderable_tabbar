# reorderable_tabbar

Reorderable TabBar package for Flutter

Requires Flutter v3.41.0-0.0.pre or higher:
https://docs.flutter.dev/release/breaking-changes/deprecate-onreorder-callback

## Example Usage

```dart
ReorderableTabBar(
  tabs: tabs,
  onReorder: (oldIndex, newIndex){},
),
```
